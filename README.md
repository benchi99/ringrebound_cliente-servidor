# Ring Rebound - Cliente y Servidor

## ESPAÑOL
En este repositorio estará el control de versiones de la primera fase del Proyecto Final de Ciclo para el Grado Superior de Desarrollo de Aplicaciones Multiplataforma.

### Cómo jugar
Tras iniciar una partida/conectarte a una partida, el jugador puede moverse con las teclas W, A, S y D, para moverte hacia delante, izquierda, detrás y derecha respectivamente. Moviendo el ratón podrás mirar a tu alrededor, y pulsando la tecla ESPACIO podrás disparar.

Tu fin es conseguir lanzar al vacío a tus contrincantes, utilizando los discos que puedes disparar. Tienes tres discos, que se regeneran con el tiempo. ¡Intenta lanzar a tantos jugadores como puedas!

### Iniciar una partida
Para iniciar una partida, haces click en el botón de Jugar, y en la sección de Crear partida, especificas un nombre de partida y el puerto en el que la partida se alojará. Cuando hagas click en Abrir servidor, el juego se iniciará, y, ¡otros jugadores podrán unirse a tu partida! Para juego en línea, se requiere que el puerto especificado esté abierto en tu router. ¡Los jugadores locales pueden unirse sin problema!

Si no se especifica ningún nombre y/o puerto, se utilizarán los siguientes valores por defecto:
* **Nombre:** Ring Rebound Server
* **Puerto:** 7777

### Conectándose a una partida
Para unirte a una partida, haces click en el botón de Jugar, y en la sección de "Conexión directa", tendrás que especificar la dirección IP y el puerto del servidor al que quieres conectarte. Cuando hagas click en "Conectar", si el servidor existe, ¡entrarás en la partida!

## ENGLISH
In this repository you will find the version control of the first phase of my Final Degree Project in Multiplatform Application Development.

### How to play
After starting a game/connecting to a server, the player can move with the W, A, S and D to move forward, to the left, backwards, and to the right respectively. Moving the mouse will allow you to look around, and pressing the SPACE key will allow you to shoot.

Your goal is to push your foes into the void, using the discs you can shoot. You have three discs, that regenerate with time. Try pushing as many players as you can!

### Starting a game
To start a game, press the button labeled "Jugar". Look then in the section called "Crear partida", and then specify a server name, and the port where the server will be listening to new player connections. Once you click in "Abrir servidor", the game will start, and other players will be able to join your game!. For Online play, it is required that the port you specified is portfowarded. Local players can join without any issue!

If a server name/server port is not specified, the following defaults will be used:
* **Name:** Ring Rebound Server
* **Port:** 7777

### Connecting to a game
To join a game, press the button labeled "Jugar". Look then in the section called "Conexión directa". Then, specify the IP Address and port where the server is being hosted. If it exists, you will join the game!
